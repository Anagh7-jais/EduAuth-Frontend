# EduAuth – AI Powered Academic Integrity & Credential Verification System

## Overview

EduAuth is an AI-powered platform designed to enhance academic integrity within educational institutions. The system integrates multiple intelligent modules to verify academic credentials, detect plagiarism, monitor exam behavior, and maintain secure academic records using blockchain verification.

The platform aims to help universities, recruiters, and administrators ensure authenticity, transparency, and trust in academic data.

---

## Problem Statement

Academic institutions face several challenges such as fake certificates, plagiarized assignments, proxy attendance, and examination malpractice. Traditional verification systems are often slow and inefficient.

EduAuth addresses these issues by providing an integrated platform that uses Artificial Intelligence and blockchain technology to verify academic credentials and monitor academic activities in a secure and transparent way.

---

## Key Features

### AI Plagiarism Detection

Detects copied or AI-generated content by comparing student submissions with reference sources and generating an originality score.

### Blockchain Certificate Verification

Stores certificate hashes securely on blockchain to prevent forgery and enable instant verification of credentials.

### Face Verified Attendance

Uses facial verification to ensure authentic student attendance during classes or examinations.

### Behavioral AI Proctoring

Monitors suspicious exam behavior such as multiple faces, unusual movements, or abnormal activities during online assessments.

### Academic Integrity Dashboard

Provides administrators with real-time metrics like Integrity Index, Originality Score, and verification logs.

### Student Academic Portfolio

Displays verified academic achievements, certificates, attendance records, and integrity scores in one unified dashboard.

## System Modules

### 1. Landing / Login Page

Role-based login interface for Admin, Student, and Recruiter with a modern user interface.

### 2. Admin Dashboard

Displays academic integrity metrics such as originality score, verification logs, alerts, and system reports.

### 3. Attendance Module

Face verification enabled attendance tracking with blockchain-based record anchoring.

### 4. Certificate Verification

Allows users to upload certificates to verify authenticity using blockchain hash verification.

### 5. AI Plagiarism Detection

Compares student submissions with reference documents and highlights copied content while generating an originality score.

### 6. Behavioral Proctoring

Monitors exam sessions and detects suspicious activities such as multiple faces or abnormal behavior.

### 7. Student Academic Portfolio

Provides a unified dashboard showing verified academic records, certificates, attendance statistics, and integrity scores.
